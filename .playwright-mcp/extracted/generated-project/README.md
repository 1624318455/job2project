# ModernWebApp

A modern, full-stack web application built with React and TypeScript, leveraging Next.js for static site generation and server-side rendering, and designed for scalability and maintainability.

## 技术栈

- React 18 + TypeScript
- Vite
- React Router

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
