# E-Commerce Product Catalog

An interactive and responsive web application that showcases a comprehensive product catalog for an e-commerce platform. The application is designed to provide a seamless user experience with advanced filtering, sorting, and search functionalities, leveraging modern JavaScript frameworks and best practices in front-end engineering.

## 技术栈

- React 18 + TypeScript
- Vite
- React Router
- 自定义Hooks
- 常用工具函数
- UI组件库

## 项目结构

```
src/
├── components/    # UI组件
├── context/       # 状态管理
├── hooks/         # 自定义Hooks
├── layouts/       # 布局组件
├── pages/         # 页面组件
├── types/         # 类型定义
├── utils/         # 工具函数
└── styles/        # 全局样式
```

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
